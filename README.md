🚀 Amazon Automation Performance Intelligence

(OpenClaw-Style Prototype)

Overview

This project is a lightweight automation prototype demonstrating how an OpenClaw-style agent workflow can support Amazon seller performance monitoring.

It simulates how manual spreadsheet analysis (PPC reports, reviews, listings) can transition into a structured, repeatable intelligence pipeline.

🏗 Tech Stack

Streamlit → Dashboard UI

n8n → Workflow orchestration

LLM → Structured reasoning & summaries

🔄 What It Demonstrates

PPC waste detection (high spend / low conversion)

Review theme clustering

Keyword opportunity identification

AI-generated performance summary

Threshold-based monitoring logic

⏱ Impact (Conceptual)

Manual review per ASIN: ~5–6 hours
Automated structured analysis: <1 minute

Primary value:

Faster detection

Standardized reporting

Reduced monitoring time

📂 Note

All datasets used are dummy/sample CSV files for simulation purposes.
No real Amazon seller data is included.
